# # 📊 Sales Analysis Project (Excel)

## 📌 Project Overview
This project focuses on analyzing sales data using Microsoft Excel to uncover meaningful insights and support data-driven decision-making. The analysis includes sales trends, product performance, and customer behavior.

## 🎯 Objectives
- Analyze overall sales performance
- Understand customer purchasing patterns
- Track monthly and regional sales trends
- Build an interactive dashboard in Excel

## 🛠️ Tools & Techniques
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Data Cleaning (Remove duplicates, handle missing values)  
- Excel Functions (SUM, AVERAGE, IF, VLOOKUP/XLOOKUP)  
- Conditional Formatting  
- Slicers & Filters  
- Dashboard Design
  
## 📁 Dataset Description
The dataset contains:
- Order ID  
- Order Date  
- Product Name / Category  
- Region  
- Sales Amount  
- Profit
- Profit Margin
  
## 🔧 Data Cleaning Steps
- Removed duplicate records  
- Handled missing/null values  
- Standardized date formats  
- Verified data consistency  

## 📈 Analysis Performed
- Total Sales and Profit Calculation  
- Monthly Sales Trend Analysis    
- Customer-wise Sales Analysis  
- Region-wise Sales Performance
- monthly Orders Count 

## 📊 Dashboard Features
- KPI Cards (Total Sales, Total Profit)  
- Monthly Sales Trend Line Chart  
- Product-wise Sales and sales Bar with profit shows line trend
- Region-wise Sales Distribution  
- Interactive Filters using Slicers  

## 💡 Key Insights
- Peak sales observed in specific months (seasonal trend)  
- A few products contribute to the majority of revenue  
- Certain regions outperform others consistently  
- detect variation of orders imact on sales

## 🚀 Learning Outcomes
- Hands-on experience with Excel data analysis  
- Strong understanding of Pivot Tables & dashboards  
- Improved data cleaning and preprocessing skills  
  

## 📌 Conclusion
This project demonstrates how Excel can be effectively used for data analysis and visualization, even without advaced tool
