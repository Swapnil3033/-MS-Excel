# # 📊 Sales Analysis Project (Excel)

## 📌 Project Overview
This project focuses on analyzing sales data using Microsoft Excel to uncover meaningful insights and support data-driven decision-making. The analysis includes sales trends, product performance, and customer behavior.

---

## 🎯 Objectives
- Analyze overall sales performance
- Identify top-selling products
- Understand customer purchasing patterns
- Track monthly and regional sales trends
- Build an interactive dashboard in Excel

---

## 🛠️ Tools & Techniques
- Microsoft Excel  
- Pivot Tables & Pivot Charts  
- Data Cleaning (Remove duplicates, handle missing values)  
- Excel Functions (SUM, AVERAGE, IF, VLOOKUP/XLOOKUP)  
- Conditional Formatting  
- Slicers & Filters  
- Dashboard Design  

---

## 📁 Dataset Description
The dataset contains:
- Order ID  
- Order Date  
- Product Name / Category  
- Customer Name  
- Region  
- Sales Amount  
- Quantity  
- Profit  

---

## 🔧 Data Cleaning Steps
- Removed duplicate records  
- Handled missing/null values  
- Standardized date formats  
- Verified data consistency  

---

## 📈 Analysis Performed
- Total Sales and Profit Calculation  
- Monthly Sales Trend Analysis  
- Top 10 Products by Sales  
- Customer-wise Sales Analysis  
- Region-wise Sales Performance  

---

## 📊 Dashboard Features
- KPI Cards (Total Sales, Total Profit, Average Sales)  
- Monthly Sales Trend Line Chart  
- Product-wise Sales Bar Chart  
- Region-wise Sales Distribution  
- Interactive Filters using Slicers
